## App URL
本アプリはAppStoreで公開されており，iOS13以降を搭載したデバイスでインストールできます．　　
https://apps.apple.com/jp/app/simpletuner/id1563149768

## Usage
1. ソースコードを置きたいディレクトリで以下のコマンドを実行  
`$ git clone git@github.com:Etsuwo/SimpleTuner.git`
2. ライブラリをインストール  
`$ pod install`
